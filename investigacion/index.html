¿Qué son los componentes pasivos en la red?

Un componente pasivo, o dispositivo pasivo, es un componente electrónico que solo puede recibir energía, que puede disipar, absorber o almacenar en un campo eléctrico o magnético. Los componentes pasivos no necesitan ningún tipo de energía eléctrica independiente para funcionar. Los dispositivos pasivos no pueden controlar la corriente mediante otra señal eléctrica.

Hay varios tipos de componentes pasivos

¿Qué son los componentes activos en la Red?

Los componentes activos en una red son aquellos dispositivos que requieren energía eléctrica para funcionar y tienen la capacidad de manejar, amplificar, y dirigir el tráfico de datos dentro de la red. Estos dispositivos no solo permiten la comunicación entre distintos dispositivos de la red, sino que también gestionan el tráfico de datos, mejoran la señal y aseguran la conectividad entre los diferentes dispositivos conectados a la red.

¿Qué es el cableado horizontal?

El cableado horizontal se refiere a los cables que se extienden desde los paneles de parcheo en las salas de telecomunicaciones hasta las estaciones de trabajo en un mismo piso. Este tipo de cableado conecta las tomas de usuario con el cableado principal de la red.

¿Qué es el cableado vertical o backbone?

El cableado vertical o backbone se refiere a la infraestructura de cableado que conecta los distintos pisos de un edificio, o incluso diferentes edificios dentro de un campus. Este cableado suele ser más robusto y soporta mayores anchos de banda, ya que es responsable de transportar el tráfico de datos entre las diferentes partes de la red.

Para qué se utiliza un cable directo en la red

Uso: Un cable directo se utiliza para conectar dispositivos diferentes entre sí, como una computadora a un switch o un router a un módem. Es el tipo de cable más común en redes domésticas y empresariales.

Para qué se utiliza un cable cruzado en la red

Uso: Un cable cruzado se utiliza para conectar dispositivos similares entre sí, como conectar directamente dos computadoras o dos switches. Este tipo de cable permite la comunicación entre dispositivos sin la necesidad de un switch o router intermediario.
